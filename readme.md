# resource allocation through history ✨

**a living poster — [live at resource.gitcoin.co](https://resource.gitcoin.co)**

> 🖼️ **like it? [buy the physical poster](https://store.gitcoin.co/products/resource-allocation-history-framed-photo-paper-poster)** — the original "resource allocation through history" print, framed, from the gitcoin store.

An interactive remake of the Gitcoin "resource allocation through history" poster — 12,000 years of deciding *who gets what, and who decides*. Two sibling posters are folded in as tabs: **power structures** and **livelihood** — from potlatch to quadratic funding, chiefdoms to forkable states, foraging to agent fleets, each one explained.

And it kept growing: **[resource.gitcoin.co/next](https://resource.gitcoin.co/next)** holds five more questions — **[what is true](https://resource.gitcoin.co/truth) · [who you are](https://resource.gitcoin.co/identity) · [who you trust](https://resource.gitcoin.co/trust) · [who keeps the record](https://resource.gitcoin.co/record) · [how you fight](https://resource.gitcoin.co/conflict)** — 313 more mechanisms, songlines to zero-knowledge proofs, wergild to peace bonds.

Click around. It sparkles.

## features

- `index.html` — three tabs: **allocation mechanisms** (81), **power structures** (80), **livelihood** (77)
- `next.html` — five tabs: **truth** (68), **identity** (66), **trust** (60), **the record** (59), **conflict** (60)
- Every mechanism explained on click — what it is, where it came from, its tradeoffs
- Each era ends with the **unbuilt ones** ✦ — specs waiting for a builder
- Every tab has its own crawlable URL + social card, so links unfurl properly
- Light & dark themes, responsive layout, reduced-motion friendly
- Self-contained HTML files — no build step, no dependencies

## running locally

It's just files. Open `index.html` or `next.html` in a browser, or serve the folder with anything:

```sh
python3 -m http.server
```
