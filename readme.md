# resource allocation through history ✨

**a living poster — [live at resource.gitcoin.co](https://resource.gitcoin.co)**

An interactive remake of the Gitcoin "resource allocation through history" poster — 12,000 years of deciding *who gets what, and who decides*. Its sibling poster, **power structures**, is folded in as a second tab: **161 mechanisms across 5 eras**, from potlatch to quadratic funding, chiefdoms to forkable states, each one explained.

Click around. It sparkles.

## features

- Two tabs: **allocation mechanisms** (81) and **power structures** (80)
- Every mechanism explained on click — what it is, where it came from, its tradeoffs
- Light & dark themes, responsive layout, reduced-motion friendly
- A single self-contained HTML file — no build step, no dependencies

## running locally

It's one file. Open `index.html` in a browser, or serve the folder with anything:

```sh
python3 -m http.server
```
